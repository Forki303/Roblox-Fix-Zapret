# Roblox-Fix-Zapret
Скрипт чтобы починить Roblox на территории РФ (zapret)
Использование:
Скачайте [Zapret](https://github.com/Flowseal/zapret-discord-youtube) последней версии. Установите этот скрипт и запустите его от имени администратора. Укажите путь к list-general в папке запрета (например: "C:\Users\Пользователь\OneDrive\Desktop\zapret-discord-youtube-1.9.3\lists\list-general.txt"). Откройте папку с zapret'ом и запустите service от имени администратора. Переключите gamefilter и ipset. Установите подходящий сервис и проверяйте.
